keyboard

/ˈkiːbɔːrd/

noun

Meaning

  • A set of keys used to operate a typewriter, computer, etc.
  • A component of musical instruments like piano, organ, harpsichord.
  • A device used to control electronic sound-producing devices.

verb

Meaning

  • To type on a computer keyboard.
